System Flow for Users and Judges

RELIEF combines triage guidance, provider discovery, and emergency response into one rural-friendly platform.

1) Symptom Intake

User enters symptoms in free text + severity and duration. Safety rules identify red flags first.

2) Explainable Triage

Output gives urgency level (Self-care / Doctor / Emergency) with clear reason statements and disclaimer.

3) Care Discovery

Nearby doctors, hospitals and pharmacies are shown with filters and quick booking actions.

4) SOS Continuity

Emergency mode shares location and fallback SMS payload for low bandwidth situations.

Architecture Diagram (in words)

  • Frontend: Web UI + maps + multilingual labels + accessibility support.
  • Decision Layer: Rule-based red-flag safety checks + NLP normalization.
  • Data Layer: User history, reminders, appointments, role-based records.
  • Notification Layer: Push + SMS fallback for reminders and SOS.
RELIEF is an assistance platform and does not replace licensed medical diagnosis.
